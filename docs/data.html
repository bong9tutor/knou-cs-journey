1. 데이터 분석과 컴퓨터

척도 4단계, 모집단/표본, 분석 워크플로우

1장은 도구 실습에 앞서 "그 도구로 무엇을 다루는지"를 정리하는 단계다. 데이터 타입(척도)을 알아야 어떤 통계량/그래프를 쓸지 정하고, 모집단과 표본을 구분해야 "내 데이터로 어디까지 말할 수 있는가"를 판단한다. 1장은 이후 모든 장의 공통 문법이다.

핵심 질문 세 가지: (1) 계산해도 되는 숫자인가, 이름표일 뿐인 숫자인가? (2) 전체인가, 일부인가(모집단 vs 표본)? (3) 무엇을 거쳐 결론에 도달하는가(워크플로우)?

측정 척도 4가지 수준

척도특징예시의미 있는 요약
명목(nominal)순서 없는 분류성별, 혈액형, 지역코드최빈값
순서(ordinal)순서는 있으나 간격 불일정만족도(상/중/하), 학점, 선호 순위중앙값까지
등간(interval)간격 일정, 절대 0 없음섭씨 온도, 연도덧셈/뺄셈(배수는 무의미)
비율(ratio)절대 0 있음, 배수 비교 가능키, 몸무게, 소득, 나이모든 사칙연산

암기 팁

명목 < 순서 < 등간 < 비율 순으로 "정보량"이 커진다. 위 단계 연산은 아래 단계에서도 항상 가능하다(비율은 다 됨).

꼭 구분할 짝

  • 변수 vs 관측치: 변수는 측정 항목(키, 성별), 관측치는 한 개체의 측정값 묶음(한 사람 = 한 행).
  • 범주형 vs 연속형: 범주형은 빈도/비율로 요약, 연속형은 평균/표준편차로 요약.
  • 모수 vs 통계량: 모집단 특성치는 모수(parameter), 표본 특성치는 통계량(statistic).
  • 기술통계 vs 추론통계: 기술통계는 가진 데이터를 요약/묘사, 추론통계는 표본으로 모집단을 추정/검정.

분석 워크플로우 7단계

문제 정의 → 데이터 수집 → 정제(전처리) → 탐색(EDA) → 분석/모형화 → 해석/시각화 → 보고

2. 데이터의 입력과 점검

결측치/이상치 처리와 Tidy Data

"쓰레기를 넣으면 쓰레기가 나온다(Garbage In, Garbage Out)." 분석가 시간의 60~80%는 모델링이 아니라 데이터를 읽어 들이고 결측치/이상치를 점검하고 정제(cleaning)하는 데 들어간다(정확한 수치라기보다 정제 비중이 그만큼 크다는 통념). 2장은 분석 전 단계의 기초 체력이다.

핵심 개념

개념설명
정형 데이터 구조행 = 관측 단위 1건, 열 = 변수 1개, 셀 = 하나의 값. 모든 분석 도구의 전제.
결측치 표기엑셀 빈 칸, R의 NA, pandas의 NaN/None. 빈칸/0/"모름" 문자열을 반드시 구분.
결측치 처리행/열 삭제(dropna, na.omit), 대치(수치형 평균/중앙값, 범주형 최빈값, 모델 예측).
이상치(outlier)분포에서 비정상적으로 먼 값. 탐지: 상자그림, IQR 규칙, Z-점수.
IQR 규칙Q1 - 1.5×IQR 미만 또는 Q3 + 1.5×IQR 초과를 이상치 후보로 본다.

Tidy Data 3원칙 (Hadley Wickham)

(1) 각 변수는 하나의 열, (2) 각 관측치는 하나의 행, (3) 각 값은 하나의 셀. "넓은 형식(wide)"을 "긴 형식(long)"으로 바꾸는 것이 핵심 변환이다.

df_clean = df[mask]

수식은 그림 붙여넣기가 아니라 반드시 수식 객체로 삽입한다. 완성한 보고서는 PDF로 내보내 표/그림 레이아웃이 깨지지 않는지 확인한다.

단순선형회귀 (추세선의 정체)

산점도의 "선형 추세선"이 곧 단순선형회귀의 결과다. 회귀식은 y = a + bx: 기울기 b는 x가 1 늘 때 y의 평균 변화량(방향/세기), 절편 a는 직선이 세로축과 만나는 위치. R제곱(결정계수)은 회귀직선이 y의 분산 중 몇 %를 설명하는지를 0~1로 나타내며, 단순선형회귀에서는 상관계수 r의 제곱과 같다.

text
=SLOPE(y범위, x범위)      기울기 b
=INTERCEPT(y범위, x범위)  절편 a
=RSQ(y범위, x범위)        R제곱

엑셀 추세선에서 [추세선 추가 → 선형] 후 "수식을 차트에 표시"와 "R-제곱 값을 차트에 표시"를 체크하면 차트에 회귀식과 R²가 함께 나온다.

R로 한 줄 회귀

fit <- lm(y ~ x, data = df)  # y를 x로 설명하는 단순선형회귀
summary(fit)                 # (Intercept)=a, x=기울기 b, Multiple R-squared=R제곱

                        중심경향 central tendency데이터가 어디쯤 모여 있는지를 요약하는 대표값. 평균/중앙값/최빈값이 3총사.
                        

                            
"대표값"이라고도 부른다. 어떤 대표값이 적절한지는 데이터의 척도와 이상치 유무에 달려 있다.

                            
예: 소득처럼 큰 값 쪽으로 꼬리가 긴 데이터는 평균보다 중앙값이 실태를 더 잘 보여 준다.

                            
연관: 

                        

                    

                    

                        평균 mean전부 더해 개수로 나눈 값. 가장 많이 쓰지만 이상치에 민감하다.
                        

                            
모든 값을 반영하는 장점이 곧 약점이다. 튀는 값 하나가 평균 전체를 끌고 간다. 위 이상치 실험실에서 직접 확인해 보자.

                            
예: 60, 70, 80, 90, 100의 평균은 80. 마지막 값이 1000이 되면 평균은 260으로 튀어 오른다.

                            
도구: 엑셀 AVERAGE · R mean() · pandas .mean()

                            
연관: 

                        

                    

                    

                        중앙값 median크기순으로 정렬했을 때 한가운데 값. 이상치가 있어도 잘 흔들리지 않는다.
                        

                            
값이 짝수 개면 가운데 두 값의 평균을 쓴다. 평균과 중앙값이 크게 다르면 분포가 한쪽으로 치우쳤다는 신호다.

                            
예: 60, 70, 80, 90, 1000의 중앙값은 여전히 80. 튀는 값 1000의 영향을 받지 않는다.

                            
도구: 엑셀 MEDIAN · R median() · pandas .median()

                            
연관: 

                        

                    

                    

                        최빈값 mode가장 자주 나오는 값. 명목척도(범주형)에서 쓸 수 있는 유일한 대표값.
                        

                            
혈액형, 시도명처럼 평균을 낼 수 없는 데이터의 대표값은 최빈값뿐이다. 두 개 이상일 수도, 없을 수도 있다.

                            
예: 70, 80, 80, 90, 100의 최빈값은 80.

                            
도구: 엑셀 MODE.SNGL(숫자 전용, 문자 범주는 COUNTIF/피벗) · R table() 최대 빈도 · pandas .mode()

                        강건성 robustness이상치가 끼어도 결과가 크게 흔들리지 않는 성질.
                        

                            
중앙값과 IQR이 대표적인 강건한 통계량이고, 평균/범위/표준편차는 민감한 쪽이다. "강건하다 = 튀는 값에 둔감하다"로 기억하면 된다.

                            
연관: 

                        

                    


                    

                        산포 dispersion데이터가 얼마나 퍼져 있는지. 중심이 같아도 퍼짐은 전혀 다를 수 있다.
                        

                            
대표값(중심)만 보면 절반만 본 것이다. 범위/분산/표준편차/IQR이 산포를 재는 도구다.

                            
예: A반(78~82)과 B반(40~120)은 평균이 80으로 같지만 퍼짐이 전혀 다르다. 위 "같은 평균, 다른 산포" 위젯 참고.

                            
연관: 

                        

                    

                    

                        범위 range최댓값 - 최솟값. 가장 단순한 산포지만 이상치 하나에 확 변한다.
                        

                            
양 끝 두 값만 쓰기 때문에 중간이 어떻게 분포하는지는 전혀 말해 주지 못한다.

                            
예: 60~100이면 범위 40. 마지막 값이 1000이면 범위는 940으로 폭발한다.

                            
도구: 엑셀 MAX - MIN · R max() - min() (range()는 최소/최대 쌍을 반환) · pandas .max() - .min()

                            
연관: 

                        

                    

                    

                        편차 deviation각 값에서 평균을 뺀 거리(값 - 평균). 그대로 다 더하면 항상 0이 된다.
                        

                            
합이 0이라 그대로는 산포 지표로 못 쓴다. 그래서 제곱해서 평균 낸 것이 분산이다.

                            
예: 평균 80일 때 60의 편차는 -20, 100의 편차는 +20. 합치면 0.

                            
연관: 

                        

                    

                    

                        분산 variance편차를 제곱해 평균 낸 값. 단위가 제곱이라 직관적으로 읽기 어렵다.
                        

                            
제곱 때문에 단위가 "점수의 제곱"처럼 이상해진다. 그래서 루트를 씌운 표준편차를 주로 읽는다. 분산이 작다 = 값들이 평균 근처에 모여 있다.

                            
예: A반(78~82)의 표본 분산은 2.5, B반(40~120)은 1000.

                            
도구: 엑셀 VAR.S/VAR.P · R var() · pandas .var()

                            
연관: 

                        

                    

                    

                        표준편차 standard deviation분산에 루트를 씌워 원래 단위로 되돌린 값. "평균에서 대략 이만큼 퍼져 있다".
                        

                            
표준편차가 작다 = 값들이 비슷하다, 크다 = 값 차이가 크다. 시험 점수라면 "몇 점 정도 퍼져 있다"로 바로 읽을 수 있는 것이 분산 대비 장점이다.

                            
예: 60, 70, 80, 90, 100의 표본 표준편차는 약 15.8점.

                            
도구: 엑셀 STDEV.S/STDEV.P · R sd() · pandas .std()

                            
연관: 

                        

                    

                    

                        사분위수 quartile정렬한 데이터를 4등분하는 세 지점. Q1(25%), Q2(50% = 중앙값), Q3(75%).
                        

                            
100등분하면 백분위수(percentile)다. Q1은 25백분위수, Q3는 75백분위수와 같다.

                            
도구: 엑셀 QUARTILE.INC · R quantile() · pandas .quantile([0.25, 0.75])

                            
연관: 

                        

                    

                    

                        IQR(사분위범위) interquartile rangeQ3 - Q1. 가운데 50%의 퍼짐이라 이상치에 강건하다.
                        

                            
이상치 판별 규칙: Q1 - 1.5×IQR 미만이거나 Q3 + 1.5×IQR 초과면 이상치 후보로 본다. 상자그림의 수염 밖 점이 바로 이것이다.

                            
예: 60, 70, 80, 90, 100에서 Q1=70, Q3=90이므로 IQR=20. 마지막 값이 1000이 되어도 IQR은 20 그대로다.

                            
연관: 

                        

                    

                    

                        n-1 보정(자유도) degrees of freedom표본의 분산/표준편차는 n이 아니라 n-1로 나눈다.
                        

                            
표본 평균을 기준으로 편차를 재면 퍼짐이 실제(모집단)보다 작게 측정되는 경향이 있어 n-1로 나눠 보정한다. 같은 데이터라도 도구/함수에 따라 결과가 미세하게 달라지는 단골 원인.

                            
도구: 엑셀 STDEV.S(n-1) vs STDEV.P(n) · R sd()는 n-1 · pandas .std()는 기본 ddof=1(n-1)

                        Z-점수 z-score(값 - 평균) / 표준편차. 평균에서 표준편차 몇 배 떨어졌는지 나타내는 표준화 점수.
                        

                            
단위가 다른 변수끼리도 비교할 수 있게 해 준다. |z|가 3 안팎을 넘으면 이상치를 의심한다.

                            
예: 평균 80, 표준편차 10인 시험에서 95점의 z = (95-80)/10 = 1.5.
