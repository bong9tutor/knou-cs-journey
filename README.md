## 📦 구성과 활용

**레포에 있는 것**

- **과목별 학습 노트** - KNOU 실제 커리큘럼을 조사·검증해 정리 (현재 1학년 8과목)
- **모바일 HTML 학습 사이트** (`docs/`) - 노트를 휴대폰으로 훑어보는 정적 사이트, GitHub Pages로 발행
- **재사용 프롬프트 모음** (`resources/프롬프트_모음.md`) - 노트 · 노티 · 노션 양식 · 사이트를 일관되게 재생성
- **텔레그램 데일리 노티** (`resources/study-noti/`) - 매일 한 단계씩 따라가는 마이크로러닝 콘텐츠
- **노션 학습 트래커** (외부) - 과목 · 회차별 진도 기록

**활용 흐름**

- **만들기**: 강의 노트 작성 -> `resources/프롬프트_모음.md`로 콘텐츠 생성 -> 텔레그램 노티 · 노션 트래커 · GitHub Pages 사이트로 발행
- **보기**: 휴대폰은 [학습 사이트](https://bong9tutor.github.io/knou-cs-journey/)(카드 · 플래시카드 · 영어 발음), 원본은 `yearN/N-S_과목명/notes/`의 마크다운
- **재생성**: 프롬프트 모음의 프롬프트를 그대로 사용 (공용 자산은 git 정본이라 `git restore docs/assets/`로 복원)
- **참고**: [1학년 학습 가이드](resources/1학년_학습_가이드.md) - 과목별 학습 전략과 무료 학습 자료

## 📁 디렉토리 구조

```
knou-cs-journey/
├── year1 ~ year4/     # 학년별. 과목 폴더 = N-S_과목명/{notes,assignments,projects}
├── docs/              # GitHub Pages 학습 사이트 (HTML + 공용 assets)
└── resources/         # 학습 가이드 · 프롬프트 모음 · 텔레그램 노티 콘텐츠
```

각 과목 폴더 하위 구성:

| 폴더 | 용도 |
|------|------|
| `notes/` | 강의 노트 · 요약 정리 |
| `assignments/` | 과제 · 출석수업 과제물(TMA) |
| `projects/` | 실습 · 프로젝트 결과물 (프로그래밍 과목만) |
