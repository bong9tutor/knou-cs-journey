## 📁 디렉토리 구조

```
knou-cs-journey/
├── year1/      # 1학년
├── year2/      # 2학년
├── year3/      # 3학년
├── year4/      # 4학년
└── resources/  # 공통 자료 (참고서적, 치트시트, 환경설정 등)
```

각 과목 폴더는 다음 하위 폴더로 구성됩니다.

| 폴더 | 용도 |
|------|------|
| `notes/` | 강의 노트, 요약 정리 |
| `assignments/` | 과제, 출석수업 과제물(TMA) |
| `projects/` | 실습 · 프로젝트 결과물 (프로그래밍 과목) |

## 🎯 졸업 전략 (1학년 신입생 · 정규 8학기 · 전 과목 수강)

- **졸업학점**: 총 130학점 (전공 51↑ · 교양 24↑ · 일반선택 나머지). 현재 49과목 약 145학점으로 충분히 초과 달성
- **수강 제한**: 학기당 최대 18학점(6과목), 직전 학기 평점 3.5↑ 등 조건 충족 시 21학점(7과목). 연 36~42학점
- **로드맵**: 학기당 6과목(18학점)씩 8학기 → 연 36학점으로 제한 내 소화. `원격대학교육의이해`(1학점·P/F)는 첫 학기에 포함
- **참고**: 평점 4.0 이상이면 130학점 도달 시점(약 7학기)에 자동 조기졸업되므로, 49과목을 끝까지 들으려면 졸업유보 신청을 고려
- 전공필수 지정 없음, 전공 초과분은 일반선택으로 인정됨. 정확한 과목별 학점은 매 학기 학교 개설일람표로 확인
