## Step 1: 컴퓨터와 프로그래밍의 이해

📘 핵심
데이터는 가공 전 원시 사실, 정보는 의미가 부여된 결과다. 컴퓨터는 입력-처리-출력 흐름으로 동작한다. 파이썬은 코드를 한 줄씩 해석하는 인터프리터 언어라 작성-실행이 빠르고 배우기 쉽다.

💡 실습
첫 프로그램을 실행해 본다.
```python
print("Hello, KNOU!")
```
출력: `Hello, KNOU!`

🎯 미션
print로 자신의 이름, 학번, "파이썬 시작!" 세 줄을 출력해 본다.

⏱️ 예상 시간: 20분

## Step 3: 변수와 기본 자료형

📘 핵심
변수는 값을 담는 이름이다. 파이썬은 동적 타입이라 자료형을 선언하지 않고 넣는 값에 따라 정해진다. 기본형은 int(정수), float(실수), str(문자열), bool(참/거짓)이며 type()으로 확인한다.

💡 실습
```python
x = 10
pi = 3.14
name = "파이썬"
print(type(x), type(pi), type(name))
```
출력: `<class 'int'> <class 'float'> <class 'str'>`

🎯 미션
정수, 실수, 문자열, 불리언 변수를 하나씩 만들고 각각의 type을 출력한다.

## Step 4: 형변환

📘 핵심
앞에서 input은 문자열만 준다고 배웠다. 계산하려면 int(), float()로 숫자로 바꿔야 한다. 반대로 str()은 숫자를 문자열로 바꾼다. int(3.9)는 반올림이 아니라 소수점을 버려 3이 된다.

💡 실습
```python

print(a + 1)
```

숫자에 10을 넣으면 출력: `11` 과 `2026년`

🎯 미션
나이를 input으로 받아 int로 바꾼 뒤, 내년 나이(나이+1)를 출력한다. int로 안 바꾸면 왜 오류가 나는지도 확인해 본다.

## Step 5: 산술 연산자

📘 핵심
사칙연산 +, -, *, / 외에 //는 몫, %는 나머지, **는 거듭제곱이다. /는 항상 실수를 주고(7/2=3.5), //는 몫만 준다(7//2=3). %는 짝수/홀수 판별에 자주 쓴다.

💡 실습
```python
print(7 / 2, 7 // 2, 7 % 2, 2 ** 10)
```

출력: `3.5 3 1 1024`

🎯 미션
숫자 두 개를 입력받아 합, 차, 곱, 몫(//), 나머지(%)를 모두 출력하는 간단 계산기를 만든다.

## Step 6: 비교와 논리 연산자

📘 핵심
비교 연산자 ==, !=, <, <=, >, >=는 참/거짓(bool)을 돌려준다. 여러 조건은 and(모두 참), or(하나라도 참), not(뒤집기)으로 묶는다. 다음 단계 조건문의 기초가 된다.

## Step 8: 문자열 메서드와 포매팅

📘 핵심
strip(공백제거), upper/lower, split(분리), join(결합), replace, find 등을 쓴다. 문자열은 불변이라 메서드는 원본을 두고 새 문자열을 반환한다. f-string으로 값을 깔끔하게 끼워 넣는다.

🎯 미션
"2026-06-08"을 split으로 연/월/일로 나눠 "2026년 6월 8일" 형태로 다시 출력한다.

## Step 9: 리스트

📘 핵심
리스트는 순서가 있고 변경 가능한 모음이다. append(추가), insert, remove, pop, sort, reverse를 쓴다. 인덱싱과 슬라이싱이 가능하고 sum/max/min도 동작한다.

🎯 미션
좋아하는 음식 5개를 리스트에 담고, 하나 추가하고 하나 제거한 뒤 가나다순으로 정렬해 출력한다.

## Step 11: 딕셔너리

📘 핵심
딕셔너리는 키-값 쌍의 모음으로 키로 값을 빠르게 찾는다. d[키]로 접근, get으로 안전 접근, keys/values/items로 순회한다. 빈도 세기에 특히 유용하다.

💡 실습
```python
student = {"name": "홍길동", "age": 23}
student["age"] = 24
for k, v in student.items():
    print(k, v)
```
출력: `name 홍길동` 과 `age 24`

🎯 미션
딕셔너리로 학생 3명의 이름과 점수를 저장하고, for문으로 "이름: 점수" 형태로 모두 출력한다.

앞에서 배운 비교/논리 연산자로 만든 조건에 따라 if, elif, else가 다른 코드를 실행한다. 파이썬은 들여쓰기(스페이스 4칸)로 블록을 구분한다.

## Step 13: for 반복문과 range

📘 핵심
for는 시퀀스의 각 요소를 순회한다. range(n)은 0부터 n-1까지 정수를 만들고, range(a, b)는 a부터 b-1까지다. enumerate는 인덱스와 값을, zip은 여러 리스트를 함께 순회한다.

💡 실습
```python
total = 0
for i in range(1, 11):
    total += i
print(total)
```
출력: `55`

🎯 미션
range를 이용해 1부터 100까지의 합을 구해 출력한다(정답은 5050).

⏱️ 예상 시간: 30분

## Step 14: while문과 break/continue

📘 핵심
while은 조건이 참인 동안 반복한다. 조건이 언젠가 거짓이 되게 해야 무한루프를 막는다. break는 반복 즉시 종료, continue는 이번 회차만 건너뛴다.

💡 실습
```python
n = 1
while n <= 5:
    if n == 3:
        n += 1
        continue
    print(n, end=" ")
    n += 1
```
출력: `1 2 4 5`

🎯 미션
while문으로 1부터 더해가다가 합이 100을 처음 넘는 순간 break로 멈추고, 그때까지 더한 마지막 수를 출력한다.

⏱️ 예상 시간: 35분

## Step 15: 중첩 반복과 구구단

📘 핵심
반복문 안에 반복문을 넣으면 표나 격자 같은 2차원 작업을 할 수 있다. 바깥 루프가 한 번 돌 때 안쪽 루프가 전부 돈다. 구구단이 대표 예제다.

💡 실습
```python
for dan in range(2, 4):
    for i in range(1, 10):
        print(f"{dan}x{i}={dan*i}", end=" ")
    print()
```
출력(첫 줄): `2x1=2 2x2=4 2x3=6 ... 2x9=18`

🎯 미션
중첩 for문으로 2단부터 9단까지 전체 구구단을 출력한다.

⏱️ 예상 시간: 30분

## Step 16: 중간 복습 (기초 종합)

📘 핵심
KNOU 강의에도 체크포인트가 있듯, 여기서 한 번 정리한다. 지금까지 배운 변수/자료형/연산자, 자료구조(리스트/튜플/집합/딕셔너리), 선택과 반복 구조를 한 문제로 묶어 손에 익힌다.

점수 5개를 리스트에 담고, for문으로 70점 이상인 점수만 골라 개수와 전체 평균을 출력한다(아직 함수 없이).

## Step 17: 컴프리헨션

📘 핵심
컴프리헨션은 반복문으로 리스트/딕셔너리/집합을 한 줄로 만든다. 형식은 [식 for 변수 in 반복 if 조건]이다. 앞 단계에서 for로 짠 필터링을 더 짧게 쓸 수 있다.

💡 실습
```python
squares = [n ** 2 for n in range(1, 6)]
evens = [n for n in range(1, 11) if n % 2 == 0]
print(squares, evens)
```
출력: `[1, 4, 9, 16, 25] [2, 4, 6, 8, 10]`

🎯 미션
1부터 30까지 중 3의 배수만 모은 리스트를 컴프리헨션으로 만든다.

## Step 19: 매개변수의 여러 형태

📘 핵심
기본값 매개변수는 인자를 생략하면 정해둔 값을 쓴다. 키워드 인자는 이름=값으로 순서 없이 전달한다. *args는 여러 개의 값을 한 묶음으로 받는다(다음 단계의 스코프와 함께 함수를 완성한다).

🎯 미션
기본값 매개변수를 가진 함수를 만들어, 인자를 생략했을 때와 줬을 때 동작이 다른 것을 확인한다.

## Step 20: 지역 변수와 전역 변수 (스코프)

📘 핵심
함수 안에서 만든 변수는 지역 변수라 함수 밖에서는 안 보인다. 함수는 바깥의 전역 변수를 읽을 수는 있지만, 값을 바꾸려면 global 선언이 필요하다(되도록 return으로 값을 돌려주는 편이 좋다).

💡 실습
```python
def make():
    x = 100   # 지역 변수
    return x
print(make())
# print(x)  # 여기서 x는 NameError
```
출력: `100`

🎯 미션
함수 안에 지역 변수를 만들고, 함수 밖에서 그 변수를 출력하려 하면 어떤 오류가 나는지 직접 확인한다.

⏱️ 예상 시간: 30분

## Step 21: 람다와 정렬

📘 핵심
람다는 이름 없는 짧은 함수다. sort나 sorted의 key 인자에 람다를 주면 원하는 기준으로 정렬할 수 있다. reverse=True로 내림차순, max/min도 key를 받는다.

💡 실습
```python
people = [("홍길동", 23), ("김철수", 19)]
people.sort(key=lambda p: p[1])
print(people)
```
출력: `[('김철수', 19), ('홍길동', 23)]`

🎯 미션
(이름, 점수) 튜플 리스트를 점수가 높은 순으로 정렬해 출력한다(key와 reverse=True 활용).

⏱️ 예상 시간: 30분

## Step 23: 파일 입출력

```python
    print(f.read())
```
출력: `첫 줄`

메모 세 줄을 파일에 쓰고, 다시 읽어 한 줄씩 strip해서 출력한다.

## Step 24: 예외처리

📘 핵심
try-except로 실행 중 오류를 잡아 프로그램이 멈추지 않게 한다. ValueError(변환 실패), ZeroDivisionError(0 나누기) 등이 흔하다. finally는 예외 여부와 무관하게 항상 실행된다.

💡 실습
```python
try:
    n = int(input("숫자: "))
    print(100 / n)
except ValueError:
    print("숫자를 입력하세요")
except ZeroDivisionError:
    print("0은 안 됩니다")
```
"abc"를 넣으면 출력: `숫자를 입력하세요`

🎯 미션
정수가 입력될 때까지 다시 묻는 안전한 입력 함수를 try-except와 while로 만든다.

⏱️ 예상 시간: 35분

## Step 25: 클래스와 객체

📘 핵심
클래스는 객체의 설계도다. __init__은 생성자, self는 객체 자신이다. 속성은 데이터, 메서드는 동작이다. 클래스로 만든 실체가 객체(인스턴스)다.

💡 실습
```python
class Dog:
    def __init__(self, name):
        self.name = name
    def bark(self):
        return f"{self.name}: 멍멍!"
print(Dog("바둑이").bark())
```
출력: `바둑이: 멍멍!`

🎯 미션
이름과 나이를 속성으로 갖고 자기소개 메서드를 가진 Student 클래스를 만들어 객체를 생성한다.

⏱️ 예상 시간: 40분

## Step 26: 상속과 다형성

📘 핵심
상속은 부모 클래스의 기능을 물려받아 확장한다. super()로 부모 생성자를 호출한다. 같은 메서드를 자식이 다르게 재정의(오버라이딩)하면, 같은 호출이 객체마다 다르게 동작한다(다형성).

💡 실습
```python
class Animal:
    def speak(self): return "..."
class Cat(Animal):
    def speak(self): return "야옹"
print(Cat().speak())
```
출력: `야옹`

🎯 미션
Animal을 상속한 Dog와 Cat을 만들어 각각 speak를 다르게 정의하고, 리스트에 담아 반복하며 호출한다.

⏱️ 예상 시간: 40분

## Step 29: 미니 프로젝트 완성 (종합)

📘 핵심
앞 단계에서 세운 설계대로 변수, 조건문, 반복문, 함수, 예외처리, 모듈을 한데 모아 작은 프로그램을 완성한다. 막히면 해당 개념을 다룬 이전 스텝으로 돌아가 확인한다.

💡 실습
숫자 맞히기 게임의 핵심 흐름이다.
```python
while True:
    try:
        guess = int(input("추측: "))
    except ValueError:
        print("정수를 입력하세요."); continue
    if guess < answer: print("더 큰 수!")
    elif guess > answer: print("더 작은 수!")
    else: print("정답!"); break
```

🎯 미션
Step 28에서 정한 프로그램을 처음부터 끝까지 직접 완성해 실행하고, 잘 동작하는지 직접 테스트한다.
