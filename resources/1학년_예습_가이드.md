# 방통대 컴퓨터과학과 1학년 예습 가이드

> 컴퓨터를 처음 다루는 신입생도 첫 학기에 흔들리지 않도록, 입학 전 4-8주 동안 무엇을 어떤 순서로 준비하면 좋은지 정리한 가이드다. 모든 자료는 무료 또는 공식 자료를 우선했고, 유료 도서는 선택 보조용으로만 표시했다.
>
> 검수 메모: 과목 배치/교재/평가 구조는 2025-2026학년도 기준 공식 정보로 확인했다. 다만 방통대는 학기마다 교재 개정, 강의 신규 제작, 과목 배치가 바뀌므로, 본격 학습 전 반드시 KNOU 컴퓨터과학과 학습정보와 수강신청 공지로 최신 정보를 다시 확인하자.

---

## 1. 개요와 예습 기본 원칙

방송통신대학교(이하 방통대)는 자기주도 학습이 전제인 원격대학이다. 강의실에서 매주 끌어주는 구조가 아니라, 영상강의 시청과 진도율 관리, 과제 마감, 기말시험 일정을 스스로 챙겨야 한다. 그래서 입학 전 예습의 진짜 목적은 "진도를 미리 빼두는 것"이 아니라, 첫 학기에 곧바로 학습 리듬을 잡을 수 있도록 워밍업하는 데 있다.

### 왜 예습하는가

- 첫 학기 6-7과목이 한꺼번에 시작되면 비전공 신입생은 용어와 도구에 압도되기 쉽다. 미리 기본 개념과 환경을 익혀두면 체감 부담이 크게 줄어든다.
- 프로그래밍 과목(파이썬, C)은 "직접 코드를 쳐본 경험"이 있고 없고의 차이가 매우 크다. 환경 설치만 미리 끝내도 첫 주가 편해진다.
- 방통대 평가 구조와 LMS 조작에 익숙하지 않아 형성평가 진도나 과제 마감을 놓치면 곧바로 점수 손실로 이어진다. 시스템 적응 자체가 중요한 예습 항목이다.

### 예습 기본 원칙

1. **전공 우선, 교양은 가볍게.** 한정된 시간은 전공/실습 과목(파이썬, C, 데이터정보처리입문, 컴퓨터과학개론)에 집중한다. 교양은 강의를 들으며 따라가도 충분하므로 예습은 용어 익히기 수준이면 된다.
2. **완벽 이해보다 전체 그림.** 예습 단계에서는 모든 내용을 정복하려 하지 말고, 큰 흐름과 핵심 용어에 익숙해지는 것을 목표로 한다. 깊은 학습은 학기 중 교재로 한다.
3. **손으로 직접 해본다.** 특히 프로그래밍과 데이터 도구(엑셀/R/Python)는 눈으로만 보지 말고 반드시 코드를 쳐보고 실행한다. 출석수업/과제가 실습형이기 때문이다.
4. **평가 구조를 먼저 이해한다.** 방통대 표준 배점은 형성평가 20점 + 중간평가 30점 + 기말시험 50점이다(2021학년도부터 적용). 중간평가는 출석수업 맞춤형평가 또는 과제물(리포트)로 이뤄진다. 이 배점에 맞춰 학습 시간을 배분하는 감각을 입학 전에 잡아둔다.

## 2. 우선순위와 학습 순서

예습 시간이 한정돼 있으므로 아래 순서를 권장한다. 프로그래밍을 가장 앞에 두는 이유는, 손에 익히는 데 시간이 가장 오래 걸리고 후속 전공의 기반이 되기 때문이다.

| 순위 | 과목 | 학기 | 성격 | 예습 비중 |
|---|---|---|---|---|
| 1 | 파이썬프로그래밍기초 | 1-1 | 전공 실습 | 높음 |
| 2 | 데이터정보처리입문 | 1-1 | 전공 실습(엑셀/R/Python) | 높음 |
| 3 | C프로그래밍 | 1-2 | 전공 실습(출석수업) | 높음 |
| 4 | 컴퓨터과학개론 | 1-2 | 전공 이론 | 중간 |
| 5 | 컴퓨터의이해 / 유비쿼터스컴퓨팅개론 / AI리터러시 | 1-1, 1-2 | 개론/소양 | 낮음 |
| 6 | 대학영어 | 1-2 | 교양(영어) | 낮음(영문법만) |
| 7 | 그 외 교양(심리학/영화/사진/정치경제/원격대학교육의이해) | 1-1, 1-2 | 교양 | 매우 낮음 |

> 주의: 위 학기 배치(특히 어떤 교양이 1학기/2학기에 열리는지)는 학번/연도에 따라 달라질 수 있다. 본인 수강신청 화면의 실제 개설 과목을 기준으로 우선순위를 조정하자.

### 권장 학습 순서

1. **프로그래밍 먼저 (파이썬 -> C).** 1학기 파이썬으로 변수/제어문/함수의 감을 잡고, 그 토대 위에서 2학기 C로 넘어간다. 파이썬에서 익힌 프로그래밍 사고가 C 학습의 진입장벽을 크게 낮춘다.
2. **데이터정보처리입문은 파이썬과 병행.** 같은 1학기 과목이라 Python 파트가 겹쳐 시너지가 크다. 단, 출석과제와 직결되는 엑셀+KOSIS 실습은 별도로 우선 챙긴다.
3. **이어서 컴퓨터과학개론.** 프로그래밍 기초가 잡힌 뒤 보면 자료구조/알고리즘 부분이 한결 수월하다. 데이터 표현(2의 보수/부동소수점)은 별도로 시간을 더 투자한다.
4. **교양은 가볍게.** 용어에 익숙해지는 1회독 수준으로 충분하다. 대학영어만 영문법 복습이 실질적으로 필요하고, 원격대학교육의이해는 신입생 적응용이라 가장 먼저 가볍게 끝낸다.

## 3. 과목별 예습 가이드

### 3.1 파이썬프로그래밍기초 (1-1, 전공 실습)

**개요/난이도.** 프로그래밍을 처음 접하는 학생을 위한 Python 기초 실습 과목이다. 컴퓨터의 이해부터 시작해 순차/선택/반복 제어구조, 함수/모듈/객체/파일, 마지막에 보드게임 만들기 실전 프로젝트까지 15주로 구성된다. 난이도 자체는 입문 수준으로 무난하지만 비교적 신설 과목이라 기출이 적어, 스스로 코드를 쳐보며 익히는 예습이 큰 도움이 된다. 비전공/노베이스라면 변수-자료형-제어문-함수까지 한 번 훑어두면 첫 학기 적응이 훨씬 수월하다.

**교재.** 정재화 교수 담당이며, 정규 교재가 2026학년도 1학기 출간 예정으로 변경되어 그 전까지는 KNOU 교안(강의안)으로 대체 수강한다. 담당 교수의 GitHub(jaehwachung/Python-Programming)에 강의 관련 실습 자료가 공개되어 있어 참고할 수 있다. 학기 직전 KNOU 공지에서 최종 교재/교안 여부를 반드시 확인하자.

**예습 핵심개념.**
- 변수와 자료형(정수/실수/문자열/불리언), 입출력(input/print)
- 연산자와 표현식, 형 변환(예: 섭씨->화씨 변환)
- 순차/선택(if-elif-else)/반복(for-while) 제어구조
- 리스트/튜플/딕셔너리 등 자료구조 기초
- 함수 정의와 호출, 매개변수와 반환값
- 모듈/패키지(import), 객체와 클래스 기초, 파일 입출력
- 실전 프로젝트(간단한 콘솔 게임) 설계와 구현
- ChatGPT 등 AI 도구 활용과 직접 작성 코드 비교(중간과제 유형)

**예습 로드맵.**
1. (감 잡기, 1-2일) 생활코딩 또는 나도코딩 기본편으로 파이썬이 무엇인지, 설치와 실행을 빠르게 훑는다. 전체 그림 파악이 목표.
2. (개발환경 세팅) Python 설치 후 'Hello, World'와 input/print를 직접 실행한다.
3. (자료형·연산, 3-4일) 점프 투 파이썬 1-2장으로 변수/자료형/형 변환을 익히고 섭씨->화씨 변환 프로그램을 직접 짠다.
4. (제어구조, 4-5일) 점프 투 파이썬 3장 + 코딩도장 연습문제로 if/for/while을 집중 연습. 점수->성적 출력, 구구단 출력을 손으로 짠다.
5. (함수·자료구조) 리스트/딕셔너리와 함수를 익히고 기능을 함수로 쪼개는 연습을 한다.
6. (모듈·객체·파일) import, 클래스 기초, 파일 읽기/쓰기를 가볍게 훑어 개념만 잡는다(학기 중 심화).
7. (미니 프로젝트) 가위바위보, 숫자 맞히기 같은 콘솔 프로그램을 처음부터 끝까지 완성한다.
8. (과목 정보 확인) 학기 전 KNOU 공지에서 2026 교재/교안, 출석수업 일정, 중간과제 공지와 AI 활용형 과제 경향을 확인한다.

**예상 시간.** 총 20-30시간(하루 1시간 기준 약 3-4주). 비전공/노베이스는 30시간 이상 권장하며, 자료형~제어구조에 시간의 약 60%를 배분하면 효율적이다.

### 3.2 데이터정보처리입문 (1-1, 전공 실습)

**개요/난이도.** 데이터 분석 기초 개념과 함께 엑셀, R, Python 세 가지 도구로 데이터를 다루는 입문 과정(전공필수 3학점, 15강)이다. 입문자 눈높이라 깊이는 얕아 난이도 자체는 낮지만, 신입생이 처음 접하는 도구가 3개나 되고 출석수업대체과제가 KOSIS(국가통계포털) 실데이터를 엑셀로 가공해 그래프를 만들고 R 코드를 작성하는 실습형이라, 손에 익히는 사전 연습이 점수와 자신감에 크게 도움이 된다. 프로그래밍 경험이 전혀 없다면 예습이 강력히 권장된다.

**교재.** 데이터정보처리입문 (이기재·김성수 공저 / 한국방송통신대학교출판문화원, 워크북 포함). E-book 약 8,650원(180일 대여), 종이책 약 17,300원. 교재는 3부 구성으로 1부 '정보와 컴퓨터', 이후 엑셀/R/Python 실습으로 이어진다.

**예습 핵심개념.**
- 데이터와 통계의 기초(변수, 척도, 모집단/표본)
- KOSIS에서 통계 검색·다운로드(엑셀/CSV)
- 엑셀 기초: 셀/수식, '숫자로 변환' 처리, 시계열 그래프
- 엑셀 기술통계와 차트(막대/꺾은선/시계열)
- R 설치(R + RStudio)와 기본 문법(변수/벡터/데이터프레임), 읽기/요약/시각화
- Python 기본 문법과 pandas 데이터프레임 조작, matplotlib 시각화
- 기술통계량(평균/분산/표준편차)과 도수분포·빈도표

### 3.3 C프로그래밍 (1-2, 전공 실습 / 출석수업)

**개요/난이도.** 2025학년도부터 1학년 2학기로 배치된 전공 실습 과목으로, 이병래 교수 담당의 출석수업(실험실습) 과목이다(멀티미디어 강의는 2025학년도 2학기에 신규 제작). C언어 문법 전반(변수/제어문/함수/포인터/구조체/파일/동적할당)을 다루는 입문 과정이며, 프로그래밍을 처음 접하는 신입생에게는 1학기 파이썬보다 체감 난이도가 높은 편이다. 특히 포인터와 메모리 개념이 어렵게 느껴지므로, 학기 전에 컴파일러 설치와 기초 문법(입출력/변수/조건·반복문)을 미리 익혀두면 출석수업과 중간평가(맞춤형평가), 기말시험을 훨씬 수월하게 따라갈 수 있다. 직접 코드를 작성해 컴파일·실행하는 경험을 미리 쌓는 것이 핵심이다.

**교재.** C프로그래밍 (김형근·곽덕훈·정재화 공저 / 한국방송통신대학교출판문화원, 2020, 336쪽, ISBN 9788920035197, 워크북 포함). 목차: 1.C언어 개요 2.자료형과 선행처리기 3.입출력함수와 연산자 4.선택/반복 제어문 5.함수와 기억클래스 6.배열과 포인터 7.구조체와 공용체 8.파일처리함수 9.메모리 동적할당.

**예습 핵심개념.**
- 개발환경 구축(Visual Studio, Dev-C++ 또는 온라인 컴파일러)과 컴파일-실행 과정
- C 프로그램 기본 구조: main 함수, #include, 세미콜론, 중괄호 블록
- 표준 입출력: printf/scanf와 서식 지정자(%d, %f, %c, %s)
- 자료형과 변수(int/float/double/char)와 형변환(캐스팅)
- 연산자(산술/관계/논리/대입/증감/비트)와 우선순위
- 선택 제어문(if-else, switch-case), 반복 제어문(for/while/do-while, break/continue)
- 함수: 정의/호출/매개변수/반환값, 기억클래스(지역/전역/static)
- 배열(1차원/다차원), 문자열(char 배열)
- 포인터: 주소(&)와 역참조(*), 포인터와 배열의 관계 (가장 중요한 난관)
- 구조체와 공용체(struct/union), 파일 입출력(fopen/fclose/fprintf/fscanf)
- 동적 메모리 할당(malloc/free)

### 3.4 컴퓨터과학개론 (1-2, 전공 이론)

**개요/난이도.** 컴퓨터과학 전반(데이터 표현, 자료구조, 알고리즘, 운영체제, 컴퓨터 구조, 프로그래밍 언어, 데이터베이스, 네트워크)을 한 학기에 훑는 전공 이론 과목이다. 분량이 매우 방대하고 "개론치고 깊다"는 후기가 많다. 각 주제가 이후 2-4학년 전공의 축소판이라 여기서 기초 감각을 잡으면 학과 전체 로드맵이 잡힌다. 특히 1장 데이터 표현(2진수/2의 보수/부동소수점 IEEE 754)과 2장 자료구조가 중간과제·기말에서 가장 어려운 부분으로 꼽혀 예습 가치가 높다.

**교재.** 컴퓨터과학개론 (이관용·정광식 공저 / 한국방송통신대학교출판문화원). 8개 장: 1.컴퓨터와 데이터 2.자료구조 3.알고리즘 4.운영체제 5.컴퓨터 구조 6.프로그래밍 언어 7.데이터베이스 8.컴퓨터 네트워크. 평가는 형성 20 + 중간 30 + 기말 50.

- 자료구조: 배열, 연결 리스트, 스택, 큐, 트리, 그래프
- 알고리즘: 정렬/탐색 기초, 시간복잡도와 Big-O
- 운영체제: 프로세스/스레드, 스케줄링, 메모리 관리, 파일 시스템
- 컴퓨터 구조: CPU 구성, 명령어 사이클, 메모리 계층, 폰 노이만 구조

## 4. 개발환경 준비

프로그래밍/데이터 과목을 위해 입학 전 아래 환경을 미리 설치해 두면 첫 주가 한결 수월하다.

### Python (파이썬프로그래밍기초 · 데이터정보처리입문)

- **설치:** [python.org](https://www.python.org/downloads/)에서 최신 안정 버전을 내려받아 설치한다. Windows 설치 시 'Add Python to PATH' 옵션을 반드시 체크한다.
- **기본 도구:** 설치 시 함께 들어오는 IDLE로 바로 시작할 수 있다.
- **권장 IDE/에디터:** Visual Studio Code(무료) + Python 확장, 또는 입문 친화적인 PyCharm Community Edition(무료). 설치가 부담되면 브라우저에서 바로 실행되는 프로그래머스 스쿨/구름EDU/Google Colab으로 시작해도 좋다.
- **데이터 분석용:** 학기 중 pandas, matplotlib가 필요하다. `pip install pandas matplotlib`로 설치한다. 설치가 번거로우면 Anaconda 배포판을 쓰면 한 번에 해결된다.

### C 컴파일러 (C프로그래밍)

- **Windows 권장:** Visual Studio Community(무료). 설치 시 'C++를 사용한 데스크톱 개발' 워크로드를 선택하면 컴파일러(MSVC)와 IDE가 함께 설치된다.
- **가벼운 대안:** Dev-C++(가볍고 입문용으로 무난) 또는 MinGW-w64로 gcc를 설치해 VS Code와 함께 사용.
- **설치 부담 없이 시작:** 구름EDU나 온라인 컴파일러(예: OnlineGDB)로 브라우저에서 바로 컴파일·실행할 수 있다. 첫 'Hello, World!'는 이렇게 시작하고, 익숙해지면 로컬 환경으로 옮기는 것을 권장한다.

### R (데이터정보처리입문)

- [CRAN](https://cran.r-project.org/)에서 R 본체를 설치하고, 이어서 [RStudio Desktop](https://posit.co/download/rstudio-desktop/)(무료)을 설치한다. RStudio가 R 코딩 환경을 훨씬 편하게 해준다.

### 권장 에디터 한눈에

| 용도 | 권장 도구 | 비고 |
|---|---|---|
| 공통 코드 에디터 | Visual Studio Code (무료) | 확장으로 Python/C 모두 지원 |

## 6. 입학 전 4-8주 예습 주차별 플랜 예시

아래는 입학 전 8주를 기준으로 한 표준 플랜이다. 시간이 4주뿐이라면 1·2·5·6주차(시스템 적응 + 파이썬 + 데이터/엑셀 + C 환경설정 핵심)만 추려서 진행한다. 1학기 과목(파이썬, 데이터정보처리입문, 1-1 교양)을 먼저, 2학기 과목(C, 컴퓨터과학개론)을 뒤에 배치했다.

| 주차 | 집중 영역 | 주요 활동 | 권장 시간 |
|---|---|---|---|
| 1주차 | 방통대 시스템 적응 | 성적평가 구조(형성20+중간30+기말50) 이해, 학사일정 달력에 기입, U-KNOU/LMS 메뉴 둘러보기, 컴퓨터과학과 교과과정 확인 | 5-7시간 |
| 2주차 | 파이썬 입문 | Python 설치 + 에디터 세팅, 나도코딩/생활코딩으로 감 잡기, 점프 투 파이썬 1-2장(변수/자료형/형변환), 섭씨->화씨 프로그램 작성 | 8-10시간 |
| 3주차 | 파이썬 심화 | 점프 투 파이썬 3장 + 코딩도장으로 제어구조(if/for/while) 집중, 함수/리스트/딕셔너리, 가위바위보 미니 프로젝트 | 8-10시간 |
| 4주차 | 데이터 도구 (엑셀+KOSIS) | 오빠두엑셀로 셀/수식/차트 기초, KOSIS에서 통계 다운로드->시계열 그래프 만들기, '숫자로 변환' 처리 연습 | 7-9시간 |
| 5주차 | 데이터 도구 (R + pandas) | R+RStudio 설치, K-MOOC R 입문(벡터/데이터프레임), pandas 기초 + matplotlib 시각화 가볍게 | 7-9시간 |
| 6주차 | C 입문 | C 컴파일러 설치(또는 구름EDU), 'Hello, World!' 컴파일·실행, 씹어먹는 C/나도코딩으로 기본 구조·입출력·자료형·연산자(교재 1-3장) | 8-10시간 |
| 7주차 | C 심화 | 제어문(if/switch/for/while), 함수와 기억클래스, 배열·문자열, 포인터 기초(가장 천천히, 영상+글 병행)(교재 4-6장) | 8-10시간 |
| 8주차 | 컴퓨터과학개론 + 교양 마무리 | CS50 Lecture 0-3, 데이터 표현(2의 보수/IEEE 754) 손계산, 자료구조 개념 훑기 / 원격대학교육의이해 시청, 대학영어 영문법 복습, 1-1 개론 교양 용어 1회독 | 8-12시간 |

> 팁: 매주 코드 과목(파이썬/C)은 반드시 직접 손으로 쳐보고, 이론/교양 과목은 용어 메모 위주로 가볍게 간다. 포인터(C)와 부동소수점(컴퓨터과학개론)에서 막히는 것은 정상이니, 예습 단계에서는 '한 번 본 적 있다' 수준이면 충분하다. 나머지 깊은 이해는 학기 중 교재와 정규 강의로 완성한다.
